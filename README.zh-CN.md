## 快速开始

### 前置依赖

在开始前，你必须先安装 Yeoman 命令行工具：

```bash
npm install -g yo
# OR
yarn global add yo
```

### 安装

```bash
npm install -g generator-antd-pro
# OR
yarn global add generator-antd-pro
```

### 创建项目

#### 完整模板

直接从 Ant Design Pro 官方仓库下载：

```bash
yo antd-pro
```

#### 简单模板

只包含简单的列表页及表单页，删除了各种冗余文件：

```bash
yo antd-pro:simple
```
